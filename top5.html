Top 5 Open-source NestJS Boilerplates to Kickstart Your Project

Explore the best free boilerplates for rapid development with NestJS

1. Brocade NextJS REST API Boilerplate

This boilerplate is a free and open-source option available on GitHub. It’s designed for rapid development with NextJS, and includes a setup for GraphQL, TypeORM, and JWT authentication.

Documentation:

An extensive, up-to-date documentation that can serve as a comprehensive guide for developers looking to build solid applications with a NextJS boilerplate.

Authentication:

  • Sign in/Sign up: Yes.
  • Social Sign-in: Yes.
  • Roles: Yes.
  • Confirm Email: Yes.
  • Forgot Password: Yes.

Config Service:

A comprehensive service, allowing environment-specific settings, which is essential when working with a NextJS boilerplate. Supports local and AWS S3 file uploads, with extensive options and supports multiple languages.

Database:

PostgreSQL + TypeORM + Knex.js

Testing:

Unit tests set up with Jest.

GraphQL Integration:

Full setup for GraphQL.

Swagger:

Integrated API documentation, making it easy to generate documentation with any NestJS boilerplate.

Docker:

Full Docker support.

2. Awesome NestJS Boilerplate

A comprehensive boilerplate option available for free on GitHub. It offers a straightforward setup with various features like authentication, configuration management, and database integration, which are crucial for developers exploring different NextJS boilerplate options.

Authentication:

  • Sign in/Sign up: Yes.
  • Social Sign-in: Yes.
  • Roles: Yes.
  • Confirm Email: No.
  • Forgot Password: No.

Config Service:

Full environment-specific support, which is beneficial when customizing a NextJS boilerplate. Supports AWS S3 (non-extendable mailing) and multiple languages.

Transactions:

Supports multiple languages.

Database:

PostgreSQL, TypeORM

Testing:

Set up for unit and e2e testing with Jest.

Swagger:

Integrated API documentation, crucial for ensuring a smooth development process with any NestJS boilerplate.

Docker:

Full Docker support.

3. NextJS Prisma Starter

A free starter template designed to integrate Prisma ORM with NextJS for efficient database management. It’s particularly useful for projects requiring a robust database layer with TypeScript support.

Documentation:

Comprehensive and up-to-date, providing clear guidance for developers looking to implement a NextJS boilerplate with Prisma.

Authentication:

  • Sign in/Sign up: Yes.
  • Social Sign-in: No.
  • Roles: Yes.
  • Confirm Email: No.
  • Forgot Password: No.

Config Service:

Full support for environment-specific settings, ideal for customizing a NextJS template.

Database:

PostgreSQL, Prisma ORM.

Testing:

Unit testing setup.

Swagger:

Integrated API documentation, useful for developers using a NextJS boilerplate to create well-documented APIs.

4. Squareboat NextJS Boilerplate

Another free starter template designed for scalable enterprise-level applications. It’s free to use and includes features like role-based access control, JWT authentication, and more.

Documentation:

Comprehensive and up-to-date, which can be valuable for developers looking to understand different NextJS boilerplate options.

Authentication:

  • Sign in/Sign up: No.
  • Social Sign-in: No.
  • Roles: No.
  • Confirm Email: No.
  • Forgot Password: No.

Config Service:

Not supported. Supports mailing with advanced configurations, which can be tailored to specific NextJS boilerplate needs. Also, supports multiple languages.

Database:

Knex.js for SQL-based databases.

Testing:

Kinley.js for SQL-based databases.

Swagger:

Not supported.

Docker:

Does not support Docker.

5. Next Hackathon Starter

This template is also open-source and free to use. It’s designed to help developers quickly bootstrap a NextJS project for hackathons, including a basic setup with MongoDB and user authentication.

Documentation:

Minimal, basic setup, which might be sufficient for quick NestJS boilerplate projects.

Authentication:

  • Sign in/Sign up: Yes.
  • Social Sign-in: No.
  • Roles: No.
  • Confirm Email: Yes.
  • Forgot Password: No.

Config Service:

Not supported. Basic mailing supported, which can be expanded in a customized NextJS boilerplate.

Database:

PostgreSQL, Prisma ORM.

Testing:

Unit tests setup.

Swagger:

Not supported.

Docker:

Does not support Docker.

Summary

Comprehensive Solutions: Brocade NextJS REST API Boilerplate and Awesome NestJS Boilerplate offer extensive features for scalable applications, including strong authentication, full documentation, and Docker support, making them excellent choices for developers working with NestJS boilerplates.

Database Management: NextJS Prisma Starter and Next Hackathon Starter are more focused on developers who need Prisma ORM, with full support for database migrations. They are particularly suitable for projects where a NextJS boilerplate with robust database management is required.

Hackathon-Ready: Next Hackathon Starter is minimalistic, ideal for quick project launches and hackathons, though it lacks some advanced features like full CI/CD integration and auto-updating dependencies, which might be necessary for more complex NextJS boilerplate projects.
